# Leetcode-style code sandboxing

Stack:
- SvelteKit
- Tailwindcss
- Docker

Main goals:
- Create a safe enviroment in which user-provided code will be able to smoothly run, without causing potential server issues
- Return console output back to the user


Future goals:
- Accounts, wchich will store user's scripts
- Leetcode-like problems
- Allowing users to access provided text files, containing problem's data
- Checking console output against provided problem answer
